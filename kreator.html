Podstawy

Czym jest kreator?

Kreator to narzędzie, które służy do łatwego tworzenia nowych obiektów bez umiejętności tworzenia stron internetowych oraz szybkiego pisania bez obaw o zepsucie strony.

Kreator automatycznie numeruje takie rzeczy jak SFA lub badania.

Podstawowe pojęcia

Pojęcia klas można znaleźć tutaj.

Inne pojęcia:

  • SFA - Save From Action - Jest to zapis, w którym jest zapisane każde działanie oraz decyzja podczas danej sytuacji. Czas, który się tam wykorzystuje, zapisuje się jako T + 0:00, który oznacza ile czasu upłynęło od rozpoczęcia sytuacji.
  • TBT - The Big Table - Najważniejsza rada w ZZS. To ona podejmuje wszystkie ważne decyzje oraz funduje ZZS.

Jak napisać?

By napisać nowy obiekt wykonaj instrukcję poniżej.

Pomysł

Zacznij od pomysłu, w którym będzie cały plan na nowy obiekt. Pamiętaj, żeby zapoznać się z Podstawowe pojęcia przed rozpoczęciem prac.

Twój pomysł musi być odpowiedni, by nie urazić nikogo lub zrobić to dla żartu z osobą, która też to uzna za śmieszny żart.

Podstawowe informacje

Na stronie kreatora uzupełnij informacje:

  • ID - Wybierz ID, które nie zostało jeszcze użyte.
  • Autor (nick) - Jeżeli masz stronę swojego profilu to wstaw tutaj swój nick. Jest to nazwa, którą jest nazwany plik twojego profilu np. https://zzswiki.github.io/profile/PieselKlif w tym przypadku nick to PieselKlif. Jak nie masz swojego profilu a nie masz umiejętności by zrobić go samemu poproś autora strony o przygotowanie takiego profilu podając mu informacje do podania.
  • Autor (wyświetlana) - Jest to nazwa, która pojawia się na samym dole artykułu obiektu.
  • Zdjęcie obiektu - Jeżeli chcesz dodać zdjęcie obiektu to ustaw tą opcję na włączoną i dodaj opis zdjęcia. Jeżeli nie to wyłącz.
  • Klasy - Ustaw klasy zgodnie z twoim obiektem.

Pisanie

Teraz dodamy więcej tekstu do artykułu. W zakładce Sekcje możesz wybrać typ sekcji oraz go dodać.

Po dodaniu sekcji masz opcję dodania do niego tekstu. Tekst to jest jedna linijka gdzie Nazwa będzie pogrubiona. Więc jeżeli chcesz normalny tekst napisać to zostaw to puste. Pamiętaj, że jeżeli piszesz wykorzystując pogrubienie Nazwy to dodawaj : np. Opis:

Jeżeli nie wiesz jak wygląda sekcja to możesz je zobaczyć w innych artykułach. (Sekcja Tekst jest najbardziej podstawowa i nie posiada żadnych efektów oprócz pogrubionej nazwy)

Ważne jest, aby zamiast znaków < oraz > używać kodu &lt; oraz &gt;, ponieważ strona traktuje to jako kawałek jej kodu.

  • < - &lt;
  • > - &gt;

Generowanie

Po zakończeniu pisania kliknij Generuj. Pojawi się kod strony oraz podgląd jak będzie to wyglądać. Możesz jeszcze wprowadzić zmiany w sekcjach oraz je zatwierdzać klikając Generuj

Jeżeli coś ci się nie podoba lub chcesz zmienić a umiesz pisać w HTML to możesz zmienić kod w output, a żeby potem zobaczyć zmiany kliknij Odśwież podgląd

Zapisanie

Zrób folder z ID twojego obiektu np. ZZS-000 i w nim utwórz plik o nazwie index.html w którym wklej kod strony twojego obiektu.

Jeżeli masz zdjęcie obiekt to w folderze obiektu zrób folder o nazwie media i w nim daj zdjęcie zmieniając mu nazwę na img1.png

Publikacja

Jest kilka opcji opublikowania artykułu:

  1. Jeżeli jesteś członkiem organizacji na GitHub
    • Za pomocą GitHub Desktop pobierz repozytorium z stroną i w plikach przenieś swój folder z obiektem do folderu obiekty. Pamiętaj by w pliku głównym index.html gdzie jest strona główna strony, dodaj swój obiekt w odpowiednim miejscu (wzoruj się innymi obiektami) z nazwą.
    • Za pomocą strony GitHub przeenieś swój folder do folderu obiekty oraz dodaj w pliku głównym index.html swój obiekt
  2. Jeżeli masz konto na github - sklonuj repozytorium strony do siebie dodając zmiany jak w punkcie 1. Jeżeli administrator zaakceptuje zmiany to twój obiekt pojawi się na stronie
  3. Discord - możesz napisać do autora strony PieselKlif na Discord wysyłając mu spakowane pliki twojego obiektu dodając w wiadomości nazwę obieektu.
