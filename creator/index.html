Docs

1. Aby zacząć pisać własny obiekt zacznij od skopiowania repo z github. Masz je naszym github. Możesz do tego użyć GitHub Desktop. Będzie łatwiej.

2. W folderze “obiekty” stwórz folder “ZZS-[ID obiektu]” i w nim główny plik strony “index.html”

Jeżeli chcesz dodać jakieś zdjęcia na stronę (np. zdjęcie obiektu) stwóż folder media i zdjęcia nazywaj “img[numer zdjęcia].png”. (zdjęcie obiektu musi miec numer 1)

3. Wypełnij kreator i naciśnij przycisk generuj.

4. Autor (nick) jest nickiem autora. Służy on do odnośnika do profilu autora np. foetus_domitor. Autor (wyświetlane) jest wyświetlany jako odnośnik do profilu autora np. Dr. Domitor. Nick ma podłogi ( _ ) zamiast spacji. Najłatwiej jest wejść na swój profil i skopiować nazwę pomiędzy /profile/ a .html.

5. Skopiuj kod html który został wygenerowany i wklej go do do “index.html ”

6. Na stronie głównej (plik “index.html” znajdujący się w głównym folderze) dodaj kolejną linijkę z linkiem do twojego artykułu (link ma mniej więcej wyglądać tak “/obiekty/ZZS-[ID]/”). Wzoruj się innymi linijkami.

7. Wyślij commit na github.

Znajomość html i tak się przydaje jeżeli chcesz robić lepszy artykuł.
